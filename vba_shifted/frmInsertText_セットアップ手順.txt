============================================================
 frmInsertText セットアップ手順
============================================================

【1】標準モジュールの追加
  VBAエディタ → 挿入 → 標準モジュール
  modInsertText.bas の内容を貼り付け

【2】UserForm の作成
  VBAエディタ → 挿入 → ユーザーフォーム
  フォーム名（プロパティ）: frmInsertText

【3】フォーム上にコントロールを配置

  ┌─────────────────────────────────────────────┐
  │  文字列追加                            [×] │
  │                                             │
  │  選択中の可視セルに文字列を追加します        │
  │                                             │
  │  追加文字列:                                │
  │  ┌─────────────────────────────────┐       │
  │  │ txtTextToAdd                     │       │
  │  └─────────────────────────────────┘       │
  │                                             │
  │  ┌─ 追加位置 (fraPosition) ──────┐        │
  │  │  (●) 先頭に追加 (optHead)      │        │
  │  │  ( ) 末尾に追加 (optTail)      │        │
  │  └────────────────────────────────┘        │
  │                                             │
  │  [✓] 二重追加をスキップ (chkSkipDuplicate) │
  │  [ ] 空白セルにも追加   (chkAddToEmpty)    │
  │                                             │
  │     [ OK ]     [ キャンセル ]               │
  └─────────────────────────────────────────────┘

【5】フォームコードの貼り付け
  frmInsertText をダブルクリック → コードウィンドウに
  frmInsertText_code.txt の内容を貼り付け

============================================================
 使い方
============================================================

1. シート上で対象セル範囲を選択（フィルタ適用中でもOK）
2. Alt+F8 →「文字列追加」を実行
3. フォームが表示される
   - 追加文字列を入力（例: "X-"）
   - 先頭 or 末尾を選択
   - 必要に応じてオプション設定
4. OK → 確認ダイアログで内容を確認 → 「はい」で実行
5. 完了メッセージに処理件数とスキップ数が表示される

============================================================
 注意事項
============================================================

- 元に戻せません（Ctrl+Z不可）。重要なデータは事前にバックアップ推奨
- 結合セルはスキップされます
- 数値・日付は内部値の文字列表現に変換されます
  例: 日付 2024/1/1 → "45292"（シリアル値）になる場合があります
  表示形式を維持したい場合は .Text プロパティの使用を検討してください
- 複数エリア選択（Ctrl+クリック）にも対応しています
- フィルタで非表示の行、手動で非表示にした行は対象外です
